## IF

```kotlin
if (a == 1) {
    print("a is equal to one.")
} else if (a == 2) {
    print("a is equal to two.")
} else {
    print("a is neither one nor two.")
}
```

---

## WHEN

```kotlin
when (it % 2) {
    0 -> print(it)
    1 -> print(it + 1)
    2 -> print(it + 2)
    3 -> print(it + 3)
}
```

## ELVIS

Эльвис-оператор действует как возвращаемое значение в случае `null`.

```kotlin
val firstName = nullName ?: "John"
```
