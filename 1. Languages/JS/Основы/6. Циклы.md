# 6. Циклы

### While

Можно так…

```javascript
let S = 0, i = 1;

while (i <= 1000) {
    S += 1 / i;
    ++i;
}

console.log(S);
```

или так…

```javascript
let S = 0, i = 1;

while ((S += i++) < 100);
console.log(S);
```

### For

Скрипт будет выглядеть так:

```javascript
let f, k = 0.5, b = 2;

for (let x = 0; x <= 1; x += 0.1) {
    f = k * x + b;
    console.log(f);
}
```

Цикл `for` можно записывать и в таком виде:

```javascript
let x = 0;

for (; x <= 1; x += 0.1) { ... }
```

или так:

```javascript
let x = 0;

for (; x <= 1;) {
    f = k * x + b;
    console.log(f);
    x += 0.1;
}
```

или, даже так:

```javascript
let x = 0;

for (;;) {
    if (x > 1) break;
    f = k * x + b;
    console.log(f);
    x += 0.1;
}
```

Или можно использовать `for` для перебора элементов:

```javascript
for (let value of fruits) {
    value = "none";
}
```

### Do while

```javascript
do {
    psw = prompt("Введите пароль", "");
} while (psw != PSW);
```

### Примечания

Также можно пользоваться операторами `continue` и `break` (в том числе `break all` - для прерывания сразу всего дерева циклов).
Использование меток:

```javascript
nextPrime: for (let i = 2; i < n; i++) {
    for (let j = 2; j < i; j++) {
        if (i % j == 0) continue nextPrime;
    }
}
```
