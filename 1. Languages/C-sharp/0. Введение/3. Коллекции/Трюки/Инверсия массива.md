# Инверсия массива

```csharp
int[] numbers = { -4, -3, -2, -1,0, 1, 2, 3, 4 };       
int n = numbers.Length; // длина массива
int k = n / 2;          // середина массива
int temp;               // вспомогательный элемент для обмена значениями

for(int i=0; i < k; i++)
{
    temp = numbers[i];
    numbers[i] = numbers[n - i - 1];
    numbers[n - i - 1] = temp;
}

```
