# Области видимости в C#

1. *Блок (в т.ч. методы)*
2. *Класс*
3. *Файл*
4. *Проект*
5. *Пространство имен*
6. *Решение*

**Правила:**
1. Переменные нижних областей видимости перекрывают собой одноименные переменные верхних областей.
2. В пределах одного контекста все имена должны быть уникальны.
