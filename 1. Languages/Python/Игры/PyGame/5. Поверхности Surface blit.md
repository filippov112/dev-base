# 5. Поверхности Surface blit

```python
surf = pygame.Surface((200, 200))  # пример создания поверхности
sc.blit(surf, (50, 50))             # нанесение поверхности surf на родительскую поверхность sc
surf.set_alpha(255)                 # задание прозрачности (видимости) поверхности (0-255) 
                                     # В том числе для всех дочерних поверхностей

# Содержимое дочерних поверхностей обрезается краями родительской

surf.fill(BLUE)                     # для анимации движения объекта перед нанесением 
surf.blit(bita, (bx, by))          # нового кадра нужно затереть старое изображение
```
