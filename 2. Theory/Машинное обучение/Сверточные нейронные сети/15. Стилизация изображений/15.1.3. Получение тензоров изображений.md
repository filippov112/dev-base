
## Получение тензоров изображений

```python
# Пропишем функцию для получения нужных нам тензоров для котиков и картины
def get_feature_representations(model):
	  style_outputs = model(x_style)		# сначала получаем четырехмерные батчи
	  content_outputs = model(x_img)        # обрезаем ненужные тензоры, и удаляем ось батчей,  оставив список 3-мерных тензоров

	  style_features = [style_layer[0] for style_layer in style_outputs[:num_style_layers]]	
	  content_features = [content_layer[0] for content_layer in content_outputs[num_style_layers:]]
	  return style_features, content_features
```
